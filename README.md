# 🏛️ Roma Explorer PWA

https://ksizorcode.github.io/roma/

¡Bienvenido a **Roma Explorer**, tu guía turística premium para la Ciudad Eterna! 🇮🇹

Esta aplicación es una **Progressive Web App (PWA)** diseñada para ofrecer una experiencia de usuario fluida, elegante y funcional, incluso sin conexión a internet.

## ✨ Características Principales

- 🏛️ **Exploración de Monumentos:** Más de 20 lugares históricos detallados.
- 🖼️ **Visuales de Alta Calidad:** Imágenes vibrantes integradas.
- 🔍 **Búsqueda y Filtros:** Encuentra lo que buscas por nombre o categoría (Cultura, Museos, Restaurantes, etc.).
- 📍 **Mapas Interactivos:** Localiza cada punto en un mapa oscuro premium de OpenStreetMap.
- 🗺️ **Navegación:** Enlaces directos a Google Maps para rutas precisas.
- ⛅ **Información Meteorológica:** Clima actual y pronóstico de 7 días.
- 📲 **Social Share:** Comparte tus lugares favoritos con amigos fácilmente.
- 🌙 **Modo Dark/Light:** Interfaz que se adapta a tu preferencia.
- 📥 **Instalable:** Úsala como una app nativa en tu móvil.

## 🚀 Cómo Empezar

1. Clona o descarga este repositorio.
2. Abre el proyecto en un servidor local (ej: `Live Server`).
3. ¡Disfruta de Roma desde la palma de tu mano!

## 🛠️ Tecnologías Usadas

- **HTML5 & CSS3:** Diseño premium con Glassmorphism.
- **JavaScript (ES6+):** Lógica modular y limpia.
- **Leaflet.js:** Para la integración de mapas.
- **Open-Meteo API:** Para datos climáticos en tiempo real.
- **Phosphor Icons:** Iconografía moderna.
